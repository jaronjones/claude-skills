## Skill Ideas

Looking for inspiration? Here are some skill ideas for this category:

- **api-docs-generator** - Generate API docs from OpenAPI specs
- **readme-builder** - Create comprehensive README files
- **changelog-from-commits** - Generate changelogs from git history
- **markdown-table-formatter** - Format and align markdown tables
- **docs-link-validator** - Check for broken links in documentation
- **diagram-from-code** - Generate diagrams from code structure
- **docstring-generator** - Generate docstrings/JSDoc

## Contributing

To add a documentation skill:

1. Create a directory: `mkdir documentation/your-skill-name`
2. Copy the template: `cp ../../SKILL_TEMPLATE.md documentation/your-skill-name/SKILL.md`
3. Fill in the details
4. Submit a PR

See [CONTRIBUTING.md](../../CONTRIBUTING.md) for full guidelines.
