## Contributing

To add an automation skill:

1. Create a directory: `mkdir automation/your-skill-name`
2. Copy the template: `cp ../../SKILL_TEMPLATE.md automation/your-skill-name/SKILL.md`
3. Fill in the details
4. Submit a PR

See [CONTRIBUTING.md](../../CONTRIBUTING.md) for full guidelines.
