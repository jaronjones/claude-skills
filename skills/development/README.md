## Contributing

To add a development skill:

1. Create a directory: `mkdir development/your-skill-name`
2. Copy the template: `cp ../../SKILL_TEMPLATE.md development/your-skill-name/SKILL.md`
3. Fill in the details
4. Submit a PR

See [CONTRIBUTING.md](../../CONTRIBUTING.md) for full guidelines.
