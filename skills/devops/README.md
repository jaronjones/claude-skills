# DevOps Skills

Skills for infrastructure, deployment, and operations.

## What's Here

Skills in this category help with:
- Container orchestration (Docker, Kubernetes)
- CI/CD pipelines (GitHub Actions, GitLab CI)
- Infrastructure as Code (Terraform, CloudFormation)
- Monitoring and observability
- Deployment automation
- Cloud platforms (AWS, GCP, Azure)

## Contributing

To add a DevOps skill:

1. Create a directory: `mkdir devops/your-skill-name`
2. Copy the template: `cp ../../SKILL_TEMPLATE.md devops/your-skill-name/SKILL.md`
3. Fill in the details
4. Submit a PR

See [CONTRIBUTING.md](../../CONTRIBUTING.md) for full guidelines.
